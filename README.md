# Learn It 🚀

An open-source, automated study companion and grade simulator engineered to optimize prep workflows for high-stakes secondary entries (King's Academy standard).

## 📂 Repository Structure
* `src/srsEngine.js` - Flashcard scheduling & spaced repetition engine
* `src/gpaTracker.js` - Weight-based grade predictor simulator
* `src/automationEngine.js` - Dynamic note & question parsing engine
* `src/inputController.js` - Central interface data router
* `src/stateManager.js` - Focus session states & metric mutations
* `src/bibliographyHelper.js` - MLA/APA scholarly citation compiler
* `src/durationLogger.js` - Task estimation accuracy variance log
* `src/burnoutRadar.js` - Continuous study time fatigue monitor
* `src/appController.js` - Unified core operational controller hub
* `src/index.js` - Complete system sandbox execution runtime
* `.github/workflows/run-tests.yml` - CI/CD automated cloud verification pipeline
